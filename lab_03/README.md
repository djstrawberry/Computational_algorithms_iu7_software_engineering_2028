# Лабораторная работа №3

Тема: построение и программная реализация алгоритма наилучшего среднеквадратичного приближения.

## Что реализовано

1. Одномерная аппроксимация табличной функции `y=f(x)` с весами `rho_i`.
   - Генерация таблицы с шумом.
   - Аппроксимация полиномами степеней `n=1,2,3`.
   - Возможность редактирования весов в консоли.
   - Отдельная демонстрация влияния весов на положение прямой (`n=1`) на одном и том же наборе точек.

2. Двумерная аппроксимация `z=f(x,y)` с весами.
   - Генерация таблицы точек `(x,y,z,rho)`.
   - Аппроксимирующие полиномы 1-й и 2-й степени.
   - Визуализация точек и поверхностей.

3. Выбор лучшей двухпараметрической нелинейной модели по минимуму среднеквадратической ошибки.
   - Используются модели:
     - `phi(x)=a*x^b`
     - `phi(x)=a*exp(b*x)`
     - `phi(x)=a+b/x`
     - `phi(x)=x/(p+q*x)`
   - Каждая модель линеаризуется и решается МНК.

4. Приближенное аналитическое решение краевой задачи для ОДУ второго порядка.
   - Использована форма `y(x)=u0(x)+sum(Ck*uk(x))`, где `u0(0)=1`, `u0(1)=0`.
   - Сравниваются результаты для `m=2` и `m=3`.

## Принятые уточнения формул

Исходный текст задания был с частично поврежденным форматированием. В коде приняты следующие интерпретации:

- Для пункта 3 используется экспоненциальная модель `a*exp(b*x)`.
- Для четвертой модели используется рациональная форма `x/(p+q*x)` как двухпараметрический вариант, линейризуемый заменой `1/y`.
- Для краевой задачи используется оператор:

`y'' + x*y' + y = x`,
`y(0)=1`, `y(1)=0`.

Если преподаватель потребует иную трактовку конкретной формулы, можно быстро заменить модель или оператор в исходниках.
